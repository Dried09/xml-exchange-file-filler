XML Filler - Заповнювач обмінних файлів
Beta

Програмне забезпечення, сукупний набір утиліт для перегляду, редагування, корегування,
збереження копій обмінних файлів XML, що використовуються в Україні з кінця 2011 року.

Software, utility for viewing, editing, correcting and saving copies of XML exchange files,
which using in Ukraine since late 2011.
